## djSystem Frontend

This folder contains the frontend portion of **djSystem**. It is intended for ongoing UI development and integration with the backend services.

### What has been developed so far

- Initial frontend project structure
- Basic application setup for future development
- Ready for component, page, and API integration work
- Foundation for styling, state handling, and routing

### How to run this part locally

1. Clone the repository:

	```bash
	git clone <repository-url>
	```

2. Move into the frontend directory:

	```bash
	cd djSystem/frontend
	```

3. Install the dependencies:

	```bash
	npm install
	```

4. Start the development server:

	```bash
	npm run dev
	```

5. Open the local URL shown in the terminal in your browser.

### Notes for further development

- Keep the backend running when working on API-based features.
- Update environment variables if your setup requires them.
- Follow the existing project structure when adding new features.
